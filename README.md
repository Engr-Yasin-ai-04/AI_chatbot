|---------|-------------|

## 🛠️ Tech Stack

### **Backend**
| Technology | Version | Purpose |
|------------|---------|---------|
| **Python** | 3.8+ | Core programming language |
| **Django** | 4.2 | Web framework |
| **TensorFlow** | 2.0 | Deep learning framework |
| **NLTK** | 3.8 | Natural Language Toolkit |
| **NumPy** | 1.24 | Numerical computations |
| **SQLite3** | - | Database |

### **Frontend**
| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure |
| **CSS3** | Styling & Animations |
| **JavaScript** | Interactivity |
| **Font Awesome** | Icons (6.4.0) |

### **ML/DL Components**
| Component | Description |
|-----------|-------------|
| **Neural Network** | 3-layer deep learning model (128-64-50) |
| **Bag of Words** | Text vectorization technique |
| **Lemmatization** | Word normalization using NLTK |
| **Intent Classification** | 50+ intent categories |
| **Confidence Threshold** | 0.25 for intent matching |
